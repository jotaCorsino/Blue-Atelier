# referencias visuais do Stitch

Esta pasta contem referencias visuais exportadas do Stitch para o Blue Atelier.

Os arquivos HTML em `html/` e as imagens em `imagens/` sao referencia visual aprovada.

O arquivo `design.md` ou `DESIGN.md` contem detalhes importantes de estilo, paleta, tipografia, espacamento, componentes e direcao visual.

Esses arquivos nao sao a implementacao final do app. Eles servem como guia visual protegido para a implementacao futura em .NET MAUI Blazor Hybrid, Razor Components e CSS proprio.

A implementacao futura deve preservar o visual aprovado. O Codex nao deve redesenhar, reinterpretar, simplificar ou alterar cores, espacamentos, cards, navegacao, filtros, listas, estados ou temas sem autorizacao explicita do usuario.
