# direção visual aprovada — blue atelier

A direção visual criada no Stitch foi aprovada como base para o Blue Atelier.

## regras principais

- manter visual moderno, minimalista, neutro e confortável;
- manter navegação lateral fixa;
- manter barra superior discreta;
- manter cards, mosaicos, bordas suaves e espaçamento generoso;
- usar azul apenas como cor de destaque;
- preservar a hierarquia visual das telas exportadas;
- preservar o estilo dos cards, botões, filtros, listas e estados;
- não transformar o app em dashboard corporativo;
- não transformar o app em e-commerce;
- não redesenhar as telas sem autorização.

## papel dos HTMLs

Os HTMLs exportados do Stitch são referência visual aprovada.

O Codex pode organizar, estudar e converter a estrutura visual para Razor Components no futuro.

O Codex não pode reinterpretar, simplificar, redesenhar ou desfigurar o visual aprovado.
